# Admin Area

مسیر پایه: `/admin/`
دسترسی: نقش Admin

---

## صفحات

### Dashboard — Index.cshtml
آمار کلی پلتفرم:
- درآمد امروز / این هفته / این ماه
- تعداد کاربران جدید
- تعداد اجراهای امروز
- تعداد ابزارهای در انتظار بررسی
- نمودار رشد کاربران ۳۰ روز
- نمودار درآمد ۳۰ روز

---

## مدیریت کاربران

### Users/Index.cshtml
- جستجو بر اساس نام، ایمیل
- فیلتر بر اساس نقش و وضعیت
- جدول: نام / ایمیل / نقش / تعداد اجرا / تاریخ ثبت / وضعیت

### Users/Detail.cshtml
- اطلاعات کامل کاربر
- تاریخچه اجراها
- تاریخچه پرداخت‌ها
- دکمه‌های عملیات: مسدود کردن / فعال کردن / تغییر نقش
- تنظیم دستی اعتبار (با ثبت دلیل)

---

## مدیریت ابزارها

### Apps/Review.cshtml — صف بررسی
- لیست ابزارهایی با Status = UnderReview
- برای هر ابزار:
  - مشاهده کامل جزئیات
  - **مشاهده پرامپت رمزگشایی شده** (فقط ادمین)
  - پیش‌نمایش فرم کاربر
  - مشاهده نمونه خروجی‌ها
- دکمه‌های عملیات: تایید / رد (با ذکر دلیل)

### Apps/Index.cshtml — همه ابزارها
- فیلتر بر اساس وضعیت، دسته‌بندی، سازنده
- عملیات: تعلیق / فعال‌سازی / ویرایش اجباری

---

## مدیریت پرداخت و مالی

### Payments/Index.cshtml — تاریخچه پرداخت‌ها
- همه تراکنش‌های پرداخت
- فیلتر بر اساس وضعیت و تاریخ
- Refund دستی

### Withdrawals/Index.cshtml — درخواست‌های برداشت
- لیست درخواست‌های در انتظار
- برای هر درخواست: مبلغ، شبا، نام سازنده
- دکمه‌های عملیات: تایید + ثبت شناسه پرداخت / رد با دلیل

### Credits/Packages.cshtml — مدیریت بسته‌های اعتباری
- افزودن / ویرایش / حذف بسته
- فعال/غیرفعال کردن
- تغییر ترتیب نمایش

---

## مدیریت AI

### AI/Providers.cshtml — مدیریت سرویس‌دهنده‌های AI
- لیست providers
- افزودن provider جدید
- ویرایش: Name، BaseUrl، API Key
- API Key با IsEncrypted ذخیره می‌شود
- دکمه "تست اتصال"

### AI/Models.cshtml — مدیریت مدل‌ها
- لیست همه مدل‌ها با فیلتر provider و capability
- افزودن مدل جدید:
  - Name، ModelId، توضیح
  - انتخاب Capabilities (چندتایی)
  - هزینه (بر اساس نوع مدل)
  - مدل پیش‌فرض برای هر capability
- فعال/غیرفعال کردن

## مدیریت محتوا

### Categories/Index.cshtml — دسته‌بندی‌ها
- افزودن / ویرایش / حذف
- Drag & Drop برای ترتیب

### Settings/Index.cshtml — تنظیمات عمومی
بخش‌بندی شده بر اساس Group:

**General:**
- نام سایت
- توضیح سایت
- ایمیل پشتیبانی
- حداقل مبلغ برداشت سازندگان

**Commission:**
- کمیسیون پیش‌فرض پلتفرم (درصد)
- کمیسیون Founding Creator

**ZarinPal:**
- MerchantId
- حالت: Sandbox / Production

**Rate Limiting:**
- حداکثر اجرا در دقیقه برای هر کاربر
- حداکثر اجرا در روز برای هر کاربر

**Registration:**
- ثبت‌نام سازنده: اتوماتیک / نیاز به تایید ادمین
